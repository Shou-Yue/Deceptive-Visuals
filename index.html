Final Reflection

To start, I wanted to write in regards to why this topic was chosen, my social media had recently been bombarded with negative news about the tariff wars from both sides, and some had almost ridiculous claims supported by visuals that were obviously full of biases (just like the ones above). Yet when viewing the feedbacks, it was very surprising how little people had picked up on these methods used. And as such, I wanted to focus this project on the same manner, and to portry how easy it was to form biased, non-ethical visualizations. With a change in scale, title, length, trends can so easily be disoriented to a desired form. For instance, there were some challenges when creating the visuals, such as null values in the dataframes or reverse trends being true, but there were so many columns that all I had to do was to loop through pairs of columns until I had found one that supported the claim. And since the columns did in fact correspond to the claim, there were no factual issues, I was not making up data nor creating false trends, only choosing my columns so that it showed a subset of what I wanted.

As of bounds regarding ethical analysis and visualizations, I can't think of any ways to limit or shape bounds, I found it surprisingly difficult to think of any hard lines not to cross, as it is more relative to each application. For instance, it is bad to make small increments in axis-scale, like I did with figure 1, but what if it is on the scale of something huge, such as global warming? Then it is clear that a small change is significant, despite it only being a tiny fraction. As such, I don't think there are any true boundaries that can be formed, and can only be judged case by case using individual morals, one visual may present itself as poorly designed to some, while others have no problems with it, and there is no real verdict on who is certainly correct in such a case, it can only be noted that each visual designer should hold themselves to a high standard, especially those who vindictivly know that what they can produce will decieve a lot of people.
